# Trabalho de Desenvolvimento Web

## Descrição do Projeto

Este projeto foi desenvolvido como parte da disciplina de Laboratório de Programação Web.
O site tem como objetivo apresentar receitas culinárias, com foco em navegação intuitiva e leitura clara dos conteúdos.

## Desenvolvedores

Este projeto foi desenvolvido em colaboração:

* **Gabriel** (https://github.com/gabrielsrfm)
* **Leosouza-devv** (https://github.com/Leosouza-devv)

## Tecnologias Utilizadas

* **HTML5**: Estruturação semântica das páginas
* **CSS3**: Estilização e layout responsivo
* **JSON**: Armazenamento de dados das receitas

## Estrutura do Repositório

* `src/pages/`: Páginas do site
* `src/styles/`: Arquivos CSS
* `public/`: Imagens e outros recursos

## Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/gabrielsrfm/Trabalho-de-Desenvolvimento-web.git
```

2. Acesse a pasta:

```bash
cd Trabalho-de-Desenvolvimento-web
```

3. Abra o arquivo `index.html` no navegador
